오늘은 가상호스트를 세팅해볼 것이다. 

# 유저 추가하기
오늘은 2개의 가상 호스트를 만들 것이기 때문에 두개의 유저를 만들것이다. 
```
user add -m <유저이름>
passwd <유저이름> 
```
명령어를 사용해서 유저를 만들어 주자.
